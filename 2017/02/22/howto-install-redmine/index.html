配置Redmine在Nginx上运行

Redmine常用的解决办法是使用passenger+nginx，但是Nginx不支持装载模块，所以需要重新编译安装Nginx，安装Nginx的Passenger有两种方式

  • 采用Passenger提供的脚本进行安装，适合新环境
  • 手动编译安装Nginx增加Passenger模块，适合已存在Nginx环境
